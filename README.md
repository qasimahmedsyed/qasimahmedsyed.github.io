# qasimahmedsyed.github.io

This repository holds the **compiled, static output** of my personal portfolio. It exists only to
serve the website through GitHub Pages at **https://qasimahmedsyed.github.io**.

The actual source code (Next.js and TypeScript) lives in a **separate private repository**. Whenever
that source is updated, a GitHub Actions workflow builds the site and publishes the generated files
here automatically. As a result, this repository is regenerated on every deploy and is not edited by
hand, so please do not open pull requests against it.

Maintained by Hafiz Syed Ahmed Qasim.
